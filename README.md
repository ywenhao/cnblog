# cnblog

复制得到：https://cdn.jsdelivr.net/gh/jquery/jquery@3.2.1/dist/jquery.min.js

gh：代表GitHub（不用改）
jquery：改为你的GitHub用户名
jquery@3.2.1：改为GitHub仓库名或者仓库发布版本号
dist：改为该仓库下的文件夹（可没有，也可多级目录）
jquery.min.js：改为该文件夹下需加速的文件（可为图片、js文件等等）
改完后复制网址就行
